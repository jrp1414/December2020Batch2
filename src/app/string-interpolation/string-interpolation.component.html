String Interpolation Component Examples

Things we can do using String Interpolation

  • 4+5 - Simple Code Expression Example - {{4+5}}
  • String Property Example : {{name}}
  • Number Property Example : {{num1+num2}}
  • Boolean Property Example : {{termsAndConditions}}
  • Custom Object Example : {{person.Name}} {{person.Address.AddLine1+" "+person.Address.City}}
  • Custom Object Example : {{person2.Name}} {{person2.Address?.AddLine1}} {{person2.Address?.City}}
  • String Method Example : {{getName()}}
